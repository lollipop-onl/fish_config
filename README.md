## Installation

Clone this repository to `~/.config/fish`

```shell
rm -rf ~/.config/fish
git clone https://github.com/lollipop-onl/fish_config.git ~/.config/fish
```

## Sync with Remote

```shell
# push to remote
fish_push

# pull from remote
fish_pull
```
